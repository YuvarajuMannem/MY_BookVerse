# 📚 MY Book Verse

A **full-stack book management web application** where users can **search, read, and manage books**, create a **wishlist**, and securely access their account.  
Built using the **MERN stack** with authentication and a modern UI.

---

# 🚀 Features

## 🔐 User Authentication
- Secure login and registration using **JWT authentication**

## 📖 Book Management
- Add books with **title, image URL, and read URL**
- View and explore books added by users

## ❤️ Wishlist
- Save books to a **personal wishlist**

## 🔍 Search Functionality
- Quickly find books using **keywords**

## 🗑 CRUD Operations
- Add, view, update, and delete books

## 🌐 Deployment
- Frontend deployed on **Vercel**
- Backend deployed on **Render**
